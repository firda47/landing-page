Portofolio & Studi Kasus

Contoh proyek yang menunjukkan pendekatan teknis, keputusan arsitektural, dan hasil bisnis nyata.

RetailX — Platform E‑Commerce

Re-architect ke microservices dan event-driven order pipeline. Hasil: throughput 4x, latency puncak turun 60%, cost per order turun 28%.

Teknologi: Kubernetes, Kafka, PostgreSQL, Prometheus.

SehatCare — Sistem Manajemen Rumah Sakit

Implementasi EMR berbasis FHIR, secure API gateway, dan audit trail. Hasil: proses klinis lebih efisien dan kepatuhan meningkat.

Teknologi: FHIR, OAuth2, ELK stack.

FinAnalytics — Platform Analitik Real-time

Streaming pipeline dan model anomali. Hasil: deteksi anomali 10x lebih cepat, false positive turun 40%.

Teknologi: Kafka, Flink, Redis, MLflow.

SmartCity — Platform Manajemen Kota Pintar

Integrasi IoT dan pipeline real-time. Hasil: respons insiden lebih cepat dan optimasi resource publik.

Teknologi: MQTT, TimescaleDB, Grafana.
